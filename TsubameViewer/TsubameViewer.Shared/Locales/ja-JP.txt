#### SettingsPage

SettingsPageTitle = 設定
AdvancedSettings = 詳細設定

SourceFoldersSettings = アプリに取り込んだフォルダの管理
RemoveSourceFolderFromApp = アプリから取り除く
ConfirmRemoveSourceFolderFromAppWithFolderName  = {0} をアプリから取り除いてもよろしいですか？
ConfirmRemoveSourceFolderFromAppDescription = この除去操作は指定フォルダやフォルダ内アイテムには一切影響しません。\n解除した指定フォルダは、いつでもアプリに再登録できます。

ThumbnailImageSettings = サムネイル表示設定
IsDisplayImageFileThubnail = 画像ファイルのサムネイルを表示
IsDisplayArchiveFileThubnail = 圧縮ファイルのサムネイルを表示
IsDisplayFolderThubnail = フォルダのサムネイルを表示
DeleteThumbnailCache = サムネイル画像のキャッシュを全て削除
ThumbnailCacheSize = 画像キャッシュ済みのストレージ容量
ProcessingThumbnailCache = 削除操作を処理中



ImageViewerSettings = 画像ビューワーの設定
IsReverseImageFliping_MouseWheel = ページ送りを逆にする（マウスホイール）
IsReverseImageFliping_Button = ページ送りを逆にする（クリック/タップ）


#### FolderListupPage

ReturnRootFolderPage = 元のフォルダへ戻る

FileSortTitle = 並べ替え
FileSortType.TitleAscending = ファイル名（昇順）
FileSortType.TitleDecending = ファイル名（降順）
FileSortType.UpdateTimeAscending = 古い順
FileSortType.UpdateTimeDecending = 新しい順
